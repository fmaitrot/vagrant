# vagrant
Setup a Java development environment in Vagrant

### Environment description
 * Ubuntu 14.04 LTS with xfce4
 * 2 vCPU
 * 4 Go RAM
 * Oracle Java 8
 * Git
 * Maven 3
 * Atom
 * IntelliJ IDEA

### Setup environment
```bash
vagrant destroy --force && vagrant up
```

### Start GUI

Connect to the VM

```bash
startxfce4&
```
